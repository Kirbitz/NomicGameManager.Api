BasicAuthenticationSecurityFilter

class BasicAuthenticationSecurityFilter(userAuthenticator: IUserAuthenticator) : GenericFilterBean

This security filter checks incoming HTTP requests for the BASIC authorization header, parses it and authenticates the credentials. Upon authentication, it constructs and adds a UsernamePasswordAuthenticationToken to the current SecurityContext. Regardless of validation success, it continues the filter chain.
